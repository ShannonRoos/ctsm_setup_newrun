# ctsm_setup_newrun
### Automization script to perform batch runs for the Heat stress Crop module model development.

- newcase: contains setup scripts to set up and submit CTSM and user_nl files to select output variables.

- HSF_lai_experiments: all CropHeatStress.F90 function adaptations to batch run differen parameterization settings for the LAI senescence experiments
- HSF_repr_experiments: all CropHeatStress.F90 and CNallocation.F90 function adaptations to run differen parameterization settings for the reproduction experinents.
